
다음 회원관리 시스템과 게시판 관리 시스템을 만들어 보아라.
DB 계정 및 비밀번호 : mboard  / 1111

MEMBER_TABLE
MEMBER_NO		NUMBER		PRIMARY KEY
MEMBER_ID		VARCHAR2(20)	
MEMBER_NAME		VARCHAR2(50)	
GENDER 			CHAR(1)		CHECK (남,여)
ENROLL_DATE		DATE		
DELETED_YN		CHAR(1)		CHECK (Y,N)

- MEMBER_NO는 시퀀스를 이용하여 신규 데이터 삽입시 자동으로 1씩 증가 (시작번호 1번부터)
- 회원 삭제시 DELETED_YN 값이 Y로 변경 되어야 함 (기본값 N)

BOARD TABLE
BOARD_NO		NUMBER		PRIMARY KEY
BOARD_CARTEGORY	VARCHAR2(20)	CHECK (공지사항, 일반글)
BOARD_TITLE		VARCHAR2(100)	NOT NULL
BOARD_CONTENT		VARCHAR2(3000)	NOT NULL
BOARD_WRITER		VARCHAR2(20)	FOREGIN KEY MEMBER (MEMBER_ID)
WRITE_DATE		DATE
DELETED_YN		CHAR(1)		

- BOARD_NO는 시퀀스를 이용하여 신규 데이터 삽입시 자동 1씩 증가 (시작번호 100번부터)
- BOARD_WRITER는 MEMBER TABLE의 MEMBER_ID를 외래키로 사용
- DELETED_YN은 삭제된 게시물일 경우 Y  / 존재하는 게시물일 경우 N 으로 저장


메인메뉴 
   1. 회원관리
   2. 게시판관리
   3. 프로그램 종료

회원관리 서브메뉴
  1. 전체회원조회
  2. 회원 아이디로 조회
  3. 회원 이름으로 조회
  4. 회원정보입력
  5. 회원정보수정
  6. 회원정보삭제
  7. 메인메뉴로


게시판 서브메뉴
  1. 게시판전체 검색
  2. 게시물 등록
  3. 작성자로 검색
  4. 제목으로 검색
  5. 게시물 수정(제목, 내용 수정 가능) 
	- 검색시 게시물 번호로 조회
  6. 게시물 삭제
  7. 메인메뉴로

※ 게시물 삭제시 게시판 전체검색에서는 보이면 안됨 (당연히 작성자, 제목 으로도 조회되면 안됨)
※ 회원정보삭제시 해당 회원이 작성한 모든 글도 삭제가 되어야 함
